Part 1 – Build a Virtual Private Cloud

Project
NextWork Cloud – AWS Networks (High Track)

Objective
Design and build a custom Amazon VPC to serve as the foundational network for future AWS resources.

What Was Built

A custom Amazon VPC with a defined CIDR block

Subnets created within the VPC

A route table to control network traffic

An Internet Gateway attached to enable internet connectivity

Key Concepts

Virtual Private Cloud (VPC)

CIDR and IP address planning

Subnets and network segmentation

Route tables and traffic flow

Internet Gateway fundamentals

Outcome
A fully functional VPC capable of hosting AWS resources and supporting both internal and internet-bound traffic. This network will be reused and extended in later labs.
